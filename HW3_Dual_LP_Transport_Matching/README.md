## Задание 1: Дуальная задача инвестиционного фонда

Рассмотрим задачу:



\[
\min_{p_1, \ldots, p_7} 100000p_1 - 20000p_4 - 20000p_5 - 20000p_6 + 50000p_7
\]



при следующих ограничениях:



\[
\begin{aligned}
p_1 + p_2 + p_3 &\geq 3.9 \\
p_2 + p_7 &\geq 1.3 \\
p_1 - 1.06p_2 + p_4 &= 0 \\
p_2 - 1.06p_3 + p_5 &= 0 \\
p_3 + p_6 &= 1.06 \\
p_{1,2,3} &\text{ — свободны} \\
p_{4,5,6} &\leq 0 \\
p_7 &\geq 0
\end{aligned}
\]



Задача не соответствует виду, который требуется для функции `linprog_simplex`.  
**Требуется:** преобразовать её и решить методом `linprog_simplex`.

## Задание 3: Парное сопоставление

Имеется некоторое количество девушек и парней. Для каждой пары известно, симпатичны они друг другу или нет.  
**Цель:** поженить максимальное количество взаимно симпатичных пар.

Запишите задачу в форме задачи **линейного программирования**.

**Подсказка:**  
- Потенциальную пару удобно обозначить бинарной переменной (0 — пара не сформирована, 1 — сформирована).  
- Так как полигамия и полиандрия не допускаются, сумма переменных для каждой девушки и каждого парня должна быть ≤ 1.
