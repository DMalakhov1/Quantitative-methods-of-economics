## 1. Преобразование и решение двойственной задачи инвестиционного фонда

Рассмотрим задачу:



\[
\min_{p_1, \ldots, p_7} \left(100000p_1 - 200000p_4 - 200000p_5 - 200000p_6 + 50000p_7\right)
\]



при условиях:



\[
\begin{aligned}
p_1 + p_2 + p_3 &\geq 3.9 \\
p_2 + p_7 &\geq 1.3 \\
p_1 - 1.06p_2 + p_4 &= 0 \\
p_2 - 1.06p_3 + p_5 &= 0 \\
p_3 + p_6 &= 1.06 \\
p_1, p_2, p_3 &\text{ — свободные переменные} \\
p_4, p_5, p_6 &\leq 0 \\
p_7 &\geq 0
\end{aligned}
\]



Задача не соответствует стандартному виду для функции `linprog_simplex`. Требуется привести её к канонической форме и решить с помощью `linprog_simplex`.

## 3. Задача о максимальном числе взаимных симпатий

Имеется некоторое количество девушек и парней. Для каждой потенциальной пары известно, симпатичны ли они друг другу.

**Цель:** поженить максимальное количество взаимно симпатичных пар.

**Формулировка в виде задачи линейного программирования:**

- Введите бинарную переменную \( x_{ij} \), равную 1, если девушка \( i \) и парень \( j \) образуют пару, и 0 — иначе.
- Учитывая, что каждый человек может быть в паре не более одного раза, для каждой девушки и каждого парня сумма соответствующих переменных должна быть ≤ 1.
- Целевая функция — максимизация суммы \( x_{ij} \) по всем взаимно симпатичным парам.

## 4. ⭐ Задача со звездочкой

Докажите справедливость выражения:



\[
\text{vec}(AXB) = (B' \otimes A) \cdot \text{vec}(X)
\]



где:
- \( \text{vec}(\cdot) \) — операция векторизации матрицы,
- \( \otimes \) — произведение Кронекера,
- \( B' \) — транспонированная матрица \( B \).
