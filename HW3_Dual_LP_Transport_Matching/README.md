## 1. Преобразование и решение двойственной задачи инвестиционного фонда

Рассмотрим задачу:

Минимизировать:  
**100000·p₁ − 200000·p₄ − 200000·p₅ − 200000·p₆ + 50000·p₇**

при условиях:

- p₁ + p₂ + p₃ ≥ 3.9  
- p₂ + p₇ ≥ 1.3  
- p₁ − 1.06·p₂ + p₄ ≥ 0  
- p₂ − 1.06·p₃ + p₅ ≥ 0  
- p₃ + p₆ ≥ 1.06  
- p₁, p₂, p₃ — свободные переменные  
- p₄, p₅, p₆ ≥ 0  
- p₇ ≥ 0

Задача не соответствует стандартному виду для функции `linprog_simplex`. Требуется привести её к канонической форме и решить с помощью `linprog_simplex`.

## 3. Задача о максимальном числе взаимных симпатий

Имеется некоторое количество девушек и парней. Для каждой потенциальной пары известно, симпатичны ли они друг другу.

**Цель:** поженить максимальное количество взаимно симпатичных пар.

**Формулировка в виде задачи линейного программирования:**

- Введите бинарную переменную xᵢⱼ, равную 1, если девушка i и парень j образуют пару, и 0 — иначе.
- Учитывая, что каждый человек может быть в паре не более одного раза, для каждой девушки и каждого парня сумма соответствующих переменных должна быть ≤ 1.
- Целевая функция — максимизация суммы xᵢⱼ по всем взаимно симпатичным парам.
